# Ferrous

The [Rust Book (mdbook)](https://doc.rust-lang.org/book/) color scheme for VS Code.

## Themes

- **Ferrous Coal** - The darkest theme with neutral gray tones (Tomorrow Night)
- **Ferrous Navy** - Deep blue-tinted dark theme
- **Ferrous Ayu** - Warm amber accents with muted backgrounds

## Installation

1. Open Extensions sidebar (`Ctrl+Shift+X`)
2. Search for "Ferrous"
3. Click Install
